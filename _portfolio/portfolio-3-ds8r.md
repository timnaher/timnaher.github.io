[DS8R_python](https://github.com/timnaher/DS8R_python) provides a Python interface for controlling the Digitimer DS8R current stimulator, commonly used in neuroscience experiments for transcutaneous electrical stimulation. This unofficial implementation makes it easy to integrate the stimulator into Python-based experimental pipelines.
